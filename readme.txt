
Projet de cour en Javascript

Installer:
	Nodejs et Npm
	Mongodb et Mongoose
	vuejs

Initialiser le projet:
	besoin de deux terminaux
	1)
		cd back/
		npm install
	2)
		cd front/
		npm install

Utiliser le projet:
	besoin de trois terminaux
	1)
		cd back/
		mongodb
	2)
		cd back/
		node indexBabel.js
	3)
		cd front/
		npm run dev
